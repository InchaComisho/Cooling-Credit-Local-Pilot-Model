# Cooling Credit Local Pilot Model: Documentation Index

[日本語](README_ja.md) | [English](README.md) | [العربية](README_ar.md)

This `docs/` directory contains implementation documents for starting Cooling Credit at a small local scale.

The purpose is to provide minimum practical procedures for measuring, cooling, recording, and connecting local actions to Local Cooling Points in schools, shopping streets, parks, farms, shelters, bus stops, public facilities, and other measurable heat-stress sites.

## Core Documents

### 1. Minimum MRV Protocol

- [MINIMUM_MRV_PROTOCOL.md](MINIMUM_MRV_PROTOCOL.md)

This document defines the minimum measurement items, recommended indicators, timing, recording method, and safety considerations for small-scale local pilots.

## Basic Flow

```text
Select a pilot site
↓
Measure baseline conditions
↓
Implement a small cooling action
↓
Measure after implementation
↓
Record photos, data, and thermal images
↓
Register a provisional Local Cooling Point
↓
Publish and share the result
```
