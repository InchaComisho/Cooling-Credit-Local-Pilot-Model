# نموذج التطبيق المحلي الصغير لأرصدة التبريد: فهرس الوثائق

[日本語](README_ja.md) | [English](README.md) | [العربية](README_ar.md)

يحتوي هذا الدليل `docs/` على وثائق تنفيذية لبدء أرصدة التبريد على نطاق محلي صغير.

الهدف هو توفير إجراءات عملية دنيا للقياس، والتبريد، والتسجيل، وربط الإجراءات المحلية بنقاط التبريد المحلية في المدارس، والشوارع التجارية، والحدائق، والمزارع، والملاجئ، ومواقف الحافلات، والمرافق العامة، وغيرها من مواقع الإجهاد الحراري القابلة للقياس.
