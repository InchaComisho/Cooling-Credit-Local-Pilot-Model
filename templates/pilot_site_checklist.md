# Small-Scale Pilot Site Checklist

## Pre-Implementation Checklist for Local Cooling Credit Pilots

This checklist is intended for schools, shopping streets, parks, farms, shelters, bus stops, public facilities, and other local sites before starting a small-scale Cooling Credit pilot.

---

## 1. Pilot Site

- [ ] The pilot site has been selected
- [ ] The intervention area has been confirmed using a map or photos
- [ ] Measurement points have been selected
- [ ] Comparison points have been selected
- [ ] Before-implementation photos have been taken
- [ ] The date for after-implementation photos has been decided

---

## 2. Measurement Preparation

- [ ] A method for measuring air temperature is available
- [ ] A method for measuring humidity is available
- [ ] A method for measuring WBGT is available
- [ ] A method for measuring surface temperature is available
- [ ] If possible, thermal imaging can be used
- [ ] Measurement times have been decided
- [ ] A person responsible for measurement has been assigned
- [ ] A recording template has been prepared

---

## 3. Intervention Details

- [ ] The cooling action has been decided
- [ ] The use of rainwater has been checked
- [ ] Water use can be recorded
- [ ] Electricity use can be recorded
- [ ] Greening area or shade area can be recorded
- [ ] If composting or humus production is included, the amount can be recorded
- [ ] If outdoor AC waste-heat mitigation is included, exhaust temperature can be recorded

---

## 4. Safety Confirmation

- [ ] The intervention does not obstruct pedestrian movement
- [ ] The intervention does not block evacuation routes
- [ ] Water does not create slippery surfaces
- [ ] Water does not contact electrical equipment
- [ ] It is confirmed that WBGT does not worsen under high humidity
- [ ] Heatstroke prevention is prioritized
- [ ] A site manager or responsible person has been assigned

---

## 5. Recording and Disclosure

- [ ] Before-implementation data has been recorded
- [ ] A plan exists to record after-implementation data
- [ ] A storage location for photos and thermal images has been decided
- [ ] Public and non-public information have been separated
- [ ] It has been decided whether to provisionally register the result as a Local Cooling Point
- [ ] The intended audience for sharing the results has been identified

---

## 6. Minimum Starting Conditions

At minimum, a small-scale pilot can begin if the following are prepared:

```text
Pilot site
Before-implementation photos
Air temperature
Humidity
WBGT
Surface temperature
After-implementation photos
After-implementation measurement
Recording sheet
```

There is no need to wait for a perfect institutional system.  
The first step is simply to measure locally, cool locally, and record the results.
