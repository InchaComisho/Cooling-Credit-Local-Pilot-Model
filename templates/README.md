# Templates Index

[Japanese](README_ja.md) | [English](README.md) | [Arabic](README_ar.md)

This directory contains practical recording templates for the Cooling Credit Local Pilot Model.

---

## CSV Templates

- [local_cooling_mrv_template.csv](local_cooling_mrv_template.csv)  
  A minimum MRV template for recording air temperature, humidity, WBGT, surface temperature, water use, rainwater use, electricity use, greening area, shade area, compost or humus volume, and related indicators.

- [cooling_point_record_sheet.csv](cooling_point_record_sheet.csv)  
  A record sheet for provisionally registering measured local results as Local Cooling Point candidates.

## Checklists

- [pilot_site_checklist_ja.md](pilot_site_checklist_ja.md)
- [pilot_site_checklist.md](pilot_site_checklist.md)
- [pilot_site_checklist_ar.md](pilot_site_checklist_ar.md)

These checklists help confirm the pilot site, measurement preparation, cooling action, safety conditions, and recording/disclosure method before starting a small-scale local pilot.

---

## How to Use

```text
1. Select a pilot site
2. Check readiness with the checklist
3. Measure baseline conditions with the MRV template
4. Implement a cooling action
5. Measure after implementation
6. Record the result in the Cooling Point sheet
7. Publish and share the result
```
