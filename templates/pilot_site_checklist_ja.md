# 小規模導入パイロットサイト・チェックリスト

## クーリングクレジット地域実証を始める前の確認項目

このチェックリストは、学校・商店街・公園・農地・避難所・バス停・公共施設などで、クーリングクレジット小規模導入を始める前に確認するためのものです。

---

## 1. 実施場所

- [ ] 実施場所を決めた
- [ ] 実施範囲を地図または写真で確認した
- [ ] 測定地点を決めた
- [ ] 比較地点を決めた
- [ ] 実施前写真を撮影した
- [ ] 実施後写真を撮影する予定日を決めた

---

## 2. 測定準備

- [ ] 気温を測る方法がある
- [ ] 湿度を測る方法がある
- [ ] WBGTを測る方法がある
- [ ] 地表温度を測る方法がある
- [ ] 可能であればサーモ画像を撮影できる
- [ ] 測定時刻を決めた
- [ ] 測定者を決めた
- [ ] 記録用テンプレートを用意した

---

## 3. 実施内容

- [ ] 冷却行動の内容を決めた
- [ ] 雨水利用の有無を確認した
- [ ] 水使用量を記録できる
- [ ] 電力使用量を記録できる
- [ ] 緑化面積または日陰面積を記録できる
- [ ] 腐葉土化・堆肥化を行う場合は量を記録できる
- [ ] 室外機排熱対策を行う場合は排気温度を記録できる

---

## 4. 安全確認

- [ ] 通行の妨げにならない
- [ ] 避難動線を妨げない
- [ ] 水で床面が滑りやすくならない
- [ ] 電気設備に水がかからない
- [ ] 高湿度時にWBGTが悪化しないか確認する
- [ ] 熱中症対策を優先する
- [ ] 管理責任者を決めた

---

## 5. 記録・公開

- [ ] 実施前データを記録した
- [ ] 実施後データを記録する予定がある
- [ ] 写真・サーモ画像の保存場所を決めた
- [ ] 公開できる情報と非公開情報を分けた
- [ ] 地域冷却ポイントとして仮記録するか決めた
- [ ] 結果を共有する相手を決めた

---

## 6. 最小開始条件

最低限、以下がそろえば小規模実証を始められる。

```text
実施場所
実施前写真
気温
湿度
WBGT
地表温度
実施後写真
実施後測定
記録表
```

完璧な制度を待つ必要はない。まず、地域で測り、冷やし、記録することから始める。
